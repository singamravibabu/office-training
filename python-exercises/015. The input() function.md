# **The input() function**

---

## 1. Basic input

```python
name = input("Enter your name: ")
```

Explanation: Pauses and waits for the user to type a name.

---

## 2. Input without a message

```python
value = input()
```

Explanation: Accepts user input without any prompt text.

---

## 3. Input for a number (still a string)

```python
num = input("Enter a number: ")
```

Explanation: Whatever the user types is stored as a string.

---

## 4. Converting input to integer

```python
age = int(input("Enter your age: "))
```

Explanation: Converts the typed value to an integer using int().

---

## 5. Converting input to float

```python
price = float(input("Enter price: "))
```

Explanation: Converts the typed value to a float (decimal).

---

## 6. Ask for two inputs

```python
a = input("Enter first value: ")
b = input("Enter second value: ")
```

Explanation: Takes two separate inputs.

---

## 7. Taking input and printing immediately

```python
text = input("Type something: ")
print(text)
```

Explanation: Shows the same text the user typed.

---

## 8. Input for a Boolean-like response

```python
ans = input("Do you agree (yes/no)? ")
```

Explanation: User may type yes or no; still treated as string.

---

## 9. Input inside a sentence

```python
city = input("Which city do you live in? ")
```

Explanation: The prompt explains what the user should type.

---

## 10. Input for list-like text

```python
items = input("Enter items separated by commas: ")
```

Explanation: User types something like: apple,banana,grapes.

---

## 11. Asking input and concatenating

```python
first = input("First name: ")
last = input("Last name: ")
print("Full name:", first + " " + last)
```

Explanation: Combines two user inputs.

---

## 12. Input inside a loop

```python
while True:
    cmd = input("Type 'exit' to stop: ")
    if cmd == "exit":
        break
```

Explanation: Repeatedly takes input until user types exit.

---

## 13. Input for password (basic)

```python
password = input("Enter password: ")
```

Explanation: Password is visible on screen (not hidden).

---

## 14. Asking numeric input and doing a calculation

```python
x = int(input("Enter a number: "))
y = int(input("Enter another number: "))
print("Sum =", x + y)
```

Explanation: Converts both inputs to integers before adding.

---

## 15. Input for a single character

```python
ch = input("Enter a single character: ")
```

Explanation: Even a single character is stored as a string.

---

## 16. Using input for conditional logic

```python
color = input("Enter color (red/blue): ")
if color == "red":
    print("You chose red.")
else:
    print("Not red.")
```

Explanation: Behavior changes based on what user typed.

---

## 17. Input for a float calculation

```python
distance = float(input("Enter distance: "))
time = float(input("Enter time: "))
print("Speed =", distance / time)
```

Explanation: Converts user input before computing.

---

## 18. Reading multiple words in one input

```python
sentence = input("Type a sentence: ")
```

Explanation: input() reads the entire line until Enter is pressed.

---

## 19. Using input with strip() (optional cleaning)

```python
name = input("Enter your name: ").strip()
```

Explanation: Removes leading/trailing spaces from what user typed.

---

## 20. Input inside a f-string message

```python
value = input(f"Enter any value: ")
```

Explanation: f-strings can be used inside input prompts.
