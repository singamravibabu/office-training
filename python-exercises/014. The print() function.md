# **The **print() function**

---

## 1. Print a simple message

```python
print("Hello, Python")
```

Explanation: Prints text to the screen.

---

## 2. Print a number

```python
print(10)
```

Explanation: Numbers can be printed directly.

---

## 3. Print multiple values

```python
print("Age:", 25)
```

Explanation: print can output several values separated by commas.

---

## 4. Print with spaces automatically added

```python
print("A", "B", "C")
```

Explanation: Commas add spaces between values.

---

## 5. Print without spaces using string concatenation

```python
print("A" + "B" + "C")
```

Explanation: Using + joins strings without spaces.

---

## 6. Print an empty line

```python
print()
```

Explanation: Calling print with no arguments prints a blank line.

---

## 7. Print using escape characters (newline)

```python
print("Line1\nLine2")
```

Explanation: \n moves text to a new line.

---

## 8. Print using a tab

```python
print("Name:\tJohn")
```

Explanation: \t adds a tab space.

---

## 9. Print quotes inside a string

```python
print("He said \"Hello\"")
```

Explanation: Escaped quotes allow quotes inside a string.

---

## 10. Print with custom separator

```python
print("A", "B", "C", sep="-")
```

Explanation: sep changes the separator between values.

---

## 11. Print with no newline at the end

```python
print("Hello", end="")
print("World")
```

Explanation: end="" prevents print from adding a newline.

---

## 12. Print with a custom ending

```python
print("Loading", end="...")
```

Explanation: end lets you add custom text after printing.

---

## 13. Print a long message using string continuation

```python
print("Python " 
      "is "
      "fun")
```

Explanation: Adjacent string literals combine automatically.

---

## 14. Print values of different data types

```python
print("Count:", 5, "Price:", 9.99)
```

Explanation: print converts values to strings automatically.

---

## 15. Print using f-strings

```python
print(f"Total = {5 + 3}")
```

Explanation: f-strings allow embedding expressions directly.

---

## 16. Print a list

```python
print([1, 2, 3])
```

Explanation: Lists can be printed directly.

---

## 17. Print a dictionary

```python
print({"a": 1, "b": 2})
```

Explanation: Dictionaries display in key-value format.

---

## 18. Print using multiline string

```python
print("""Line A
Line B
Line C""")
```

Explanation: Triple quotes allow printing multi-line text.

---

## 19. Print a repeated pattern

```python
print("*" * 10)
```

Explanation: Repetition creates simple text patterns.

---

## 20. Print Unicode characters

```python
print("Unicode test: ₹, ✓, Ω")
```

Explanation: print supports Unicode output.
