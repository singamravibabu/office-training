# **Compound Assignment Operators in Python**

Compound assignment operators include:
`+=`  `-=`  `*=`  `/=`  `//=`  `%=`  `**=`  `&=`  `|=`  `^=`  `>>=`  `<<=`

Each example shows how the operator updates the variable.

---

## 1. Addition assignment

```python
x = 10
x += 5
```

Explanation: x becomes 15.

---

## 2. Subtraction assignment

```python
y = 20
y -= 8
```

Explanation: y becomes 12.

---

## 3. Multiplication assignment

```python
a = 4
a *= 3
```

Explanation: a becomes 12.

---

## 4. Division assignment

```python
b = 25
b /= 5
```

Explanation: b becomes 5.0.

---

## 5. Floor division assignment

```python
c = 27
c //= 4
```

Explanation: c becomes 6.

---

## 6. Modulus assignment

```python
d = 27
d %= 4
```

Explanation: d becomes 3.

---

## 7. Exponent assignment

```python
e = 3
e **= 4
```

Explanation: e becomes 81.

---

## 8. Add negative number

```python
f = 10
f += -3
```

Explanation: f becomes 7.

---

## 9. Subtract negative number

```python
g = 10
g -= -5
```

Explanation: g becomes 15.

---

## 10. Multiply by negative number

```python
h = 6
h *= -2
```

Explanation: h becomes -12.

---

## 11. Bitwise AND assignment

```python
i = 6
i &= 3
```

Explanation: i becomes 2 (binary AND).

---

## 12. Bitwise OR assignment

```python
j = 6
j |= 3
```

Explanation: j becomes 7 (binary OR).

---

## 13. Bitwise XOR assignment

```python
k = 6
k ^= 3
```

Explanation: k becomes 5 (binary XOR).

---

## 14. Right shift assignment

```python
l = 16
l >>= 2
```

Explanation: l becomes 4.

---

## 15. Left shift assignment

```python
m = 5
m <<= 1
```

Explanation: m becomes 10.

---

## 16. Combined multiplication after assignment

```python
n = 2
n *= 5
n *= 2
```

Explanation: n becomes 20.

---

## 17. Combined addition and subtraction

```python
p = 10
p += 3
p -= 1
```

Explanation: p becomes 12.

---

## 18. Using exponent twice

```python
q = 2
q **= 3
q **= 2
```

Explanation: q becomes 64.

---

## 19. Using modulus after subtraction

```python
r = 17
r -= 5
r %= 4
```

Explanation: r becomes 0.

---

## 20. Using mixed arithmetic operators

```python
s = 50
s /= 2
s -= 5
s *= 3
```

Explanation: s becomes 75.0.
