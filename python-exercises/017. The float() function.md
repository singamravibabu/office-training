# **The float() function**

The **float() function** converts values into floating-point (decimal) numbers.

---

## 1. Convert an integer to float

```python
print(float(10))
```

Explanation: Converts 10 into 10.0.

---

## 2. Convert a numeric string to float

```python
print(float("3.14"))
```

Explanation: Converts the string into a decimal number.

---

## 3. Convert a whole-number string

```python
print(float("20"))
```

Explanation: Becomes 20.0.

---

## 4. Convert a negative string number

```python
print(float("-5.5"))
```

Explanation: Negative values convert normally.

---

## 5. Convert a float-like string with spaces

```python
print(float("   7.25   "))
```

Explanation: Leading/trailing spaces are ignored.

---

## 6. Convert scientific-notation string

```python
print(float("1e3"))
```

Explanation: “1 × 10³” becomes 1000.0.

---

## 7. Convert integer in scientific float form

```python
print(float(2e2))
```

Explanation: 2e2 = 200.0.

---

## 8. Convert a boolean True

```python
print(float(True))
```

Explanation: True converts to 1.0.

---

## 9. Convert a boolean False

```python
print(float(False))
```

Explanation: False converts to 0.0.

---

## 10. Convert result of integer division

```python
print(float(5 // 2))
```

Explanation: 5//2 = 2, then converted to 2.0.

---

## 11. Convert result of normal division

```python
print(float(10 / 4))
```

Explanation: Already a float, but float() still returns a float.

---

## 12. Convert complex notation error example (real part only possible)

```python
print(float(7))
```

Explanation: Only real numbers can be converted with float() directly.

---

## 13. Convert a string with plus sign

```python
print(float("+9.5"))
```

Explanation: Plus sign is allowed.

---

## 14. Convert a float written as int

```python
print(float("15"))
```

Explanation: Numeric string converts to 15.0.

---

## 15. Convert large integer to float

```python
print(float(999999999999))
```

Explanation: Very large integers convert but may lose precision.

---

## 16. Convert a float result from an expression

```python
print(float(3 * 2.5))
```

Explanation: Expression result is converted again.

---

## 17. Convert a string with newline

```python
print(float("8.75\n"))
```

Explanation: float() ignores newline at the end.

---

## 18. Convert a boolean expression result

```python
print(float(5 > 3))
```

Explanation: 5 > 3 is True, converted to 1.0.

---

## 19. Convert a hexadecimal integer to float

```python
print(float(0xA))
```

Explanation: 0xA = 10 in decimal, so float is 10.0.

---

## 20. Convert an octal integer to float

```python
print(float(0o12))
```

Explanation: 0o12 (octal) = 10, converted to 10.0.
