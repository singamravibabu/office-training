# **Line Continuation Character (`\`) in Python**

The line-continuation character **`\`** lets Python treat multiple lines as one logical line.

---

## 1. Break a long addition into two lines

```python
x = 10 + \
    20
```

Explanation: The backslash continues the expression on the next line.

---

## 2. Break multiplication across lines

```python
y = 5 * \
    4
```

Explanation: Python reads this as 5 * 4.

---

## 3. Continue a string literal (concatenation by adjacency)

```python
text = "Hello " \
       "World"
```

Explanation: Two string literals next to each other combine automatically.

---

## 4. Continue a list across lines

```python
numbers = [1, 2, 3, \
           4, 5, 6]
```

Explanation: The list continues after the backslash.

---

## 5. Continue a tuple across lines

```python

          3, 4)
```

Explanation: Tuple values split into multiple lines.
