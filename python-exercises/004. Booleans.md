# **Boolean values in Python**

---

## 1. Print True

```python
print(True)
```

Explanation: True is a built-in Boolean value in Python.

---

## 2. Print False

```python
print(False)
```

Explanation: False is the second Boolean value.

---

## 3. Print two Boolean values together

```python
print(True, False)
```

Explanation: Multiple literals can be printed in a single statement.

---

## 4. Print True inside a sentence

```python
print("The value is", True)
```

Explanation: Booleans can appear alongside strings in print.

---

## 5. Print False inside a sentence

```python
print("The result is", False)
```

Explanation: Booleans can be displayed as part of output messages.

---

## 6. Print True as part of a list literal

```python
print([True])
```

Explanation: Lists can contain Boolean values.

---

## 7. Print False inside a list literal

```python
print([False])
```

Explanation: A list can also contain False.

---

## 8. Print both True and False inside a list

```python
print([True, False])
```

Explanation: Multiple Boolean values can be placed in one list.

---

## 9. Print a tuple with a Boolean

```python
print((True,))
```

Explanation: Tuples can hold Boolean values.

---

## 10. Print a tuple with both values

```python
print((True, False))
```

Explanation: Both Boolean values can exist together in a tuple.

---

## 11. Print a dictionary with Boolean value

```python
print({"active": True})
```

Explanation: Dictionaries can store Boolean literals as values.

---

## 12. Print a dictionary containing False

```python
print({"enabled": False})
```

Explanation: False can also be used as a dictionary value.

---

## 13. Print a list mixing Booleans and other literals

```python
print(["Start", True, "End"])
```

Explanation: Booleans can be mixed with other literal types.

---

## 14. Print a nested structure using Booleans

```python
print({"status": True, "flags": [False, True]})
```

Explanation: Booleans can appear in nested data structures.

---

## 15. Print repeated Boolean literals

```python
print(True, True, False, True)
```

Explanation: Repeating Boolean literals is allowed; they are still just True or False.
