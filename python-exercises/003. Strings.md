# **Strings in Python**

---

## 1. Print a simple string

```python
print("Hello")
```

Explanation: A string is text inside quotes.

---

## 2. Print a single character

```python
print("A")
```

Explanation: A string can contain just one character.

---

## 3. Print an empty string

```python
print("")
```

Explanation: An empty string contains no characters.

---

## 4. Print a sentence

```python
print("Python programming is interesting")
```

Explanation: Strings can contain multiple words.

---

## 5. Print a string with spaces at the start

```python
print("   spaced")
```

Explanation: Spaces inside the quotes are part of the string.

---

## 6. Print a string with spaces at the end

```python
print("end   ")
```

Explanation: Trailing spaces are also part of the string.

---

## 7. Print a string containing digits

```python
print("12345")
```

Explanation: Even numbers in quotes are treated as text.

---

## 8. Print a string containing symbols

```python
print("#$%&@")
```

Explanation: Strings can contain any characters placed inside quotes.

---

## 9. Print a string with mixed characters

```python
print("A1b2C3")
```

Explanation: Strings can mix letters, digits, and symbols.

---

## 10. Print a string with a newline escape sequence

```python
print("Line1\nLine2")
```

Explanation: Escape sequences allow controlling formatting inside a string.

---

## 11. Print a string with a tab

```python
print("Name\tAge")
```

Explanation: \t represents a tab space.

---

## 12. Print a string containing quotes

```python
print("He said, \"Hello\"")
```

Explanation: Quotes inside a string must be escaped.

---

## 13. Print a multiline string

```python
print("""This is
a multiline
string""")
```

Explanation: Triple quotes allow multiple lines.

---

## 14. Print a raw string

```python
print(r"C:\folder\new")
```

Explanation: Raw strings treat backslashes as literal characters.

---

## 15. Print multiple string literals together

```python
print("Python", "is", "fun")
```

Explanation: print can display several string literals separated by commas.
