# **The int() function**

The **int() function** converts values into integers whenever possible.

---

## 1. Convert a string number to int

```python
print(int("10"))
```

Explanation: Converts the string "10" into the integer 10.

---

## 2. Convert a float to int

```python
print(int(3.14))
```

Explanation: Removes the decimal part (floor toward zero), result is 3.

---

## 3. Convert a negative float

```python
print(int(-4.99))
```

Explanation: Converts to -4 (decimal removed).

---

## 4. Convert a positive float with .0

```python
print(int(15.0))
```

Explanation: Converts smoothly since 15.0 represents an exact integer.

---

## 5. Convert a numeric string with spaces

```python
print(int("   25   "))
```

Explanation: Leading/trailing spaces are allowed.

---

## 6. Convert a string with a plus sign

```python
print(int("+40"))
```

Explanation: int() accepts explicit + sign.

---

## 7. Convert a string with a minus sign

```python
print(int("-20"))
```

Explanation: int() converts negative numbers correctly.

---

## 8. Convert True to int

```python
print(int(True))
```

Explanation: True becomes 1.

---

## 9. Convert False to int

```python
print(int(False))
```

Explanation: False becomes 0.

---

## 10. Convert a float result from an expression

```python
print(int(10 / 3))
```

Explanation: 10/3 is a float; int() removes decimal part.

---

## 11. Convert integer inside parentheses

```python
print(int((5)))
```

Explanation: Parentheses do not affect conversion.

---

## 12. Convert string digits from user-like input

```python
print(int("123"))
```

Explanation: Valid numeric strings convert easily.

---

## 13. Convert binary string to int (base 2)

```python
print(int("1010", 2))
```

Explanation: Converts binary 1010 to decimal 10.

---

## 14. Convert hexadecimal string to int (base 16)

```python
print(int("A", 16))
```

Explanation: Hex A equals decimal 10.

---

## 15. Convert octal string to int (base 8)

```python
print(int("12", 8))
```

Explanation: Octal 12 equals decimal 10.

---

## 16. Convert string with newline

```python
print(int("30\n"))
```

Explanation: Newline at end is acceptable.

---

## 17. Convert float in scientific notation

```python
print(int(1.9e2))
```

Explanation: 1.9e2 = 190.0 â†’ int becomes 190.

---

## 18. Convert a very large numeric string

```python
print(int("999999999999"))
```

Explanation: Python can convert very large integers without issue.

---

## 19. Convert integer written as string with underscore

```python
print(int("1_000"))
```

Explanation: Underscores are allowed inside numeric strings.

---

## 20. Convert boolean expression result

```python
print(int(5 > 3))
```

Explanation: 5 > 3 evaluates to True, which converts to 1.
