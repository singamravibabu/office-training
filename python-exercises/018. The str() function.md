# **The str() function**

The **str() function** converts values into **strings** (text).

---

## 1. Convert an integer to a string

```python
print(str(10))
```

Explanation: 10 becomes "10" (text).

---

## 2. Convert a float to a string

```python
print(str(3.14))
```

Explanation: Converts 3.14 into the string "3.14".

---

## 3. Convert a boolean True to string

```python
print(str(True))
```

Explanation: Becomes "True".

---

## 4. Convert boolean False to string

```python
print(str(False))
```

Explanation: Becomes "False".

---

## 5. Convert a list to string

```python
print(str([1, 2, 3]))
```

Explanation: List becomes the string "[1, 2, 3]".

---

## 6. Convert a tuple to string

```python
print(str((4, 5, 6)))
```

Explanation: Tuple becomes "(4, 5, 6)".

---

## 7. Convert a dictionary to string

```python
print(str({"a": 1, "b": 2}))
```

Explanation: Produces "{"a": 1, "b": 2}"".

---

## 8. Convert a set to string

```python
print(str({1, 2, 3}))
```

Explanation: Produces a string like "{1, 2, 3}".

---

## 9. Convert None to string

```python
print(str(None))
```

Explanation: None becomes "None".

---

## 10. Convert a complex number

```python
print(str(2 + 3j))
```

Explanation: Produces the string "(2+3j)".

---

## 11. Convert the result of an expression

```python
print(str(5 + 7))
```

Explanation: 12 becomes the string "12".

---

## 12. Convert the result of a comparison

```python
print(str(5 > 3))
```

Explanation: True becomes "True".

---

## 13. Convert an empty string (still string)

```python
print(str(""))
```

Explanation: Produces "" because it is already a string.

---

## 14. Convert a number inside print with text

```python
print("Age: " + str(25))
```

Explanation: str() allows concatenation with other strings.

---

## 15. Convert a float in scientific notation

```python
print(str(1.2e3))
```

Explanation: Prints "1200.0".

---

## 16. Convert a large integer

```python
print(str(99999999999999))
```

Explanation: Large integer converted safely to text.

---

## 17. Convert a boolean expression with numbers

```python
print(str(10 == 10))
```

Explanation: True becomes "True".

---

## 18. Convert list with mixed types

```python
print(str([1, "A", 3.5, False]))
```

Explanation: Entire list becomes one string.

---

## 19. Convert a multi-line string using str()

```python
print(str("""Line1
Line2"""))
```

Explanation: str() keeps the multi-line structure.

---

## 20. Convert repeated pattern to string

```python
print(str("*" * 5))
```

Explanation: The pattern "*****" becomes a string (already is).
